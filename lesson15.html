Lesson 15: Python Vocabulary

Learn the key terms and concepts every Python programmer should know.

Variable
A name that refers to a value stored in memory. Variables store data you can manipulate.
Data Types
The kind of data a variable holds, e.g., int (integer), float (decimal numbers), str (strings/text), bool (True/False).
List
An ordered collection of values, which can be changed. Example: my_list = [1, 2, 3].
Tuple
An ordered collection of values that cannot be changed (immutable). Example: my_tuple = (1, 2, 3).
Dictionary
A collection of key-value pairs. Example: my_dict = {"name": "Alex", "age": 25}.
Function
A reusable block of code that performs a task. Defined with def. Example:
def greet(name): print("Hello,", name)
Loop
A control structure for repeating code. Common loops: for and while.
Conditional
Code that runs only if a condition is true, using if, elif, and else.
Class
A blueprint for creating objects (OOP). Defines attributes and methods.
Object
An instance of a class with specific data.
Module
A Python file with reusable code that can be imported.
Exception
An error that occurs during program execution, which can be handled using try and except.
Indentation
Whitespace at the start of a line that defines code blocks in Python. Proper indentation is required.
Comment
Text in the code ignored by Python, used for notes. Starts with #.
Boolean
A data type with only two values: True or False.
Operator
Symbols that perform operations on variables and values, like +, -, *, /, and comparison operators like ==.
