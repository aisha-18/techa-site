Lesson 1: Intro to Python

Learn the basics of Python — perfect for beginners and future tech leaders.

Why Learn Python?

Python is one of the most popular and beginner-friendly programming languages. It’s used by developers at Google, NASA, and startups all over the world.

First Python Code

print("Hello, world!")

This simple line prints text to the screen. It’s often your first step into the world of programming.

Using Comments

# Integer

age = 14

# Float

temperature = 36.6

# Boolean

is_coding_fun = True

# String

name = "Techa"

# List

colors = ["red", "blue", "green"]

# Tuple

coordinates = (10, 20)

# Set

unique_numbers = {1, 2, 3}

# Dictionary

student = {
    "name": "Alex",
    "age": 14,
    "grade": "A"
}
