Lesson 9: Dictionaries

Learn how to store data as key-value pairs using Python dictionaries.

What is a Dictionary?

A dictionary stores data in pairs: keys and values. It's unordered and keys must be unique.

# Creating a dictionary
person = {
    "name": "Alex",
    "age": 25,
    "city": "New York"
}
print(person)
print(person["name"])    # Alex

Adding & Updating Items

You can add new key-value pairs or update existing ones:

person["email"] = "alex@example.com"
person["age"] = 26
print(person)

Looping Through a Dictionary

Use a for loop to get keys and values:

for key in person:
    print(key, "->", person[key])

# Or use items() method
for key, value in person.items():
    print(key, "->", value)
